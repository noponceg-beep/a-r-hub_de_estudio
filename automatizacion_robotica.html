Automatización Industrial & Robótica
¿Qué es la Automatización Industrial?
La Automatización Industrial es la tecnología que permite controlar procesos o sistemas productivos con la mínima participación humana. Optimiza recursos, mejora la calidad y aumenta la seguridad en entornos industriales.
Objetivos Principales
  • Económico: Reducir costos de producción y operación.
  • Calidad: Mejorar la uniformidad y precisión de productos.
  • Seguridad: Evitar que trabajadores realicen tareas peligrosas o repetitivas.
  • Productividad: Aumentar la producción sin comprometer la calidad.
  • Conectividad: Obtener información en tiempo real para tomar mejores decisiones.
Definiciones Clave
Sistema: Combinación de componentes que actúan juntos para lograr un objetivo. Tiene entradas, un proceso de transformación y salidas.
Control Automático: Tecnología que permite regular procesos sin intervención humana continua, mediante dispositivos electrónicos, mecánicos o informáticos.
Nota importante: La automatización no busca reemplazar completamente al humano, sino liberar al trabajador de tareas repetitivas o peligrosas para que se enfoque en actividades de mayor valor.
// CONCEPTOS CLAVE
Fundamentos teóricos de la automatización
Sistema de Control
Conjunto de dispositivos que regulan, dirigen o administran el comportamiento de otros dispositivos para lograr un resultado deseado.
Lazo de Control
Estructura que permite medir, comparar y ajustar continuamente una variable para mantenerla en el valor deseado.
Setpoint
Valor de referencia que se desea mantener en un sistema de control automático.
Variable Controlada
Magnitud física que se mide y regula en un proceso (temperatura, presión, nivel, caudal, etc.).
Cadena de Control Automático
La cadena de control sigue este orden secuencial:
  1. Medir: Sensores captan la variable del proceso.
  2. Transmitir: La señal se envía al controlador.
  3. Controlar: El controlador procesa y decide la acción correctiva.
  4. Actuar: Actuadores modifican el proceso según la orden del controlador.
Tipos de Control
Control en Lazo Abierto: No hay retroalimentación. El sistema ejecuta una acción sin verificar el resultado.
Control en Lazo Cerrado: Incluye retroalimentación. El sistema mide continuamente la salida y ajusta la acción para mantener el setpoint.
// COMPONENTES
Elementos fundamentales de un sistema automatizado
Sensor / Transductor
Dispositivo que convierte una magnitud física (temperatura, presión, distancia) en una señal eléctrica medible. Es el "sentido" del sistema.
Ejemplos: Termocupla (temperatura), sensor de presión, encoder (posición), fotocélula (luz).
Controlador
Cerebro del sistema. Recibe señales de los sensores, las procesa según un algoritmo o programa, y envía órdenes a los actuadores.
Ejemplos: PLC (Controlador Lógico Programable), microcontrolador, sistema SCADA.
Actuador
Elemento que ejecuta la acción física ordenada por el controlador. Transforma señales eléctricas en movimiento, calor, presión, etc.
Ejemplos: Motor eléctrico, electroválvula, cilindro neumático, resistencia de calentamiento.
Interfaz Hombre-Máquina (HMI)
Pantalla o panel que permite al operador visualizar el estado del proceso e interactuar con el sistema (cambiar parámetros, iniciar/detener procesos).
Componente Función Ejemplo
Sensor Medir Termocupla
Transmisor Enviar señal Cable, red industrial
Controlador Procesar y decidir PLC, microcontrolador
Actuador Ejecutar acción Motor, válvula
// TECNOLOGÍAS
Herramientas y sistemas de automatización
Lógica Cableada
Sistema de control basado en componentes físicos como relés, contactores y temporizadores. No utiliza software; el comportamiento se define por el cableado físico.
Ventajas: Simple, robusto en ambientes sencillos.
Desventajas: Difícil de modificar, requiere recablear para cambios.
Lógica Programable (PLC)
Controlador industrial programable que reemplaza la lógica cableada por software. Permite reprogramar el sistema sin cambios de hardware.
Ventajas: Flexible, fácil de reprogramar, escalable.
Ejemplos: Siemens S7, Allen-Bradley, Schneider Electric Modicon.
SCADA (Supervisory Control and Data Acquisition)
Sistema de supervisión y adquisición de datos. Permite monitorear y controlar procesos industriales desde una estación central, generalmente con interfaz gráfica.
Usos: Plantas de manufactura, redes eléctricas, tratamiento de aguas.
Robótica Industrial
Aplicación de robots programables para realizar tareas repetitivas con alta precisión: soldadura, pintura, ensamblaje, paletizado.
Características: Alta velocidad, repetibilidad, capacidad de trabajar 24/7.
Industria 4.0
Integración de sistemas ciber-físicos, IoT, inteligencia artificial y análisis de datos en tiempo real para crear fábricas inteligentes.
Internet de las Cosas (IoT)
Conexión de sensores y dispositivos a internet para recopilar y analizar datos en tiempo real, optimizando procesos industriales.
// CLASE 1: CONCEPTOS BÁSICOS
Automatización tradicional e Industria 4.0
Concepto de Automatización Industrial
Corresponde al área técnica que tiene por objetivo controlar procesos de producción, empleando para ello conocimientos eléctricos, electrónicos, mecánicos, informáticos y de proyectos. El objetivo es poder integrar equipos automáticos (también llamados "automatismos") para operar procesos industriales con mínima intervención humana, mejorando la calidad y la productividad.
Definición de Sistema de Control Automatizado
Corresponde a la integración de componentes de medición, control, transmisión, fuerza y comunicación, con el fin de interactuar con un proceso físico y modificar su comportamiento según los requerimientos del proceso productivo (temperatura, nivel, flujo, peso, etc.). Un sistema automatizado integra elementos electrónicos, eléctricos, mecánicos, hidráulicos, neumáticos e informáticos.
Objetivos de la Automatización Industrial
  • Disminuir costos: Humanos, energéticos, materiales (El dinero manda).
  • Mejorar la calidad: Producto acabado uniforme y consistente.
  • Seguridad: Evitar tareas difíciles, peligrosas, rápidas o repetitivas.
  • Conectividad: Información en tiempo real para mejores decisiones.
  • Flexibilidad: Aumentar producción y adaptación al mercado.
Tecnologías para Automatizar
Lógica Cableada: Sistema de control basado en conexiones físicas entre componentes eléctricos (relés, contactores, temporizadores). No utiliza software; el comportamiento se define por el cableado físico. Su modificación requiere cambiar el cableado, por lo que es menos flexible.
Lógica Programada (PLC): Sistema basado en programación mediante controladores electrónicos (PLC - Controladores Lógicos Programables). Permite controlar procesos mediante software, facilitando modificar o ampliar el sistema sin cambiar el cableado físico. Se utiliza en procesos automatizados más complejos.
Control Automático en Lazo Cerrado
El control automático es el mantenimiento de un valor deseado (nivel, temperatura, flujo, etc.), midiendo el valor existente, comparándolo con el valor deseado y utilizando la diferencia para corregir dicha condición. Funciona sin intervención humana y requiere:
  • Equipo de medida: Capaz de medir una variable física.
  • Transmisor: Transforma la medición en señal eléctrica proporcional.
  • Indicador: Pantalla que muestra el valor medido.
  • Unidad de control: Cerebro del sistema (PLC).
  • Actuador: Ejecuta acciones físicas.
Proceso a Controlar
Es la actividad que se desea controlar. Ejemplo: Controlar el llenado del estanque del sistema de agua.
Variable Controlada
Variable física específica que deseo mantener en un valor. Ejemplo: NIVEL del estanque.
Variable Manipulada
Variable sobre la cual se actúa para mantener la variable controlada. Ejemplo: FLUJO de entrada mediante bomba.
Set Point (SP)
Valor deseado donde se quiere mantener la variable controlada. Ejemplo: 500 litros (capacidad del estanque).
Industria 4.0
La Industria 4.0 o Cuarta Revolución Industrial es la transformación digital de los procesos industriales mediante la integración de tecnologías avanzadas para crear fábricas y sistemas más automatizados y conectados.
Características principales:
  • Internet de las Cosas (IoT): Máquinas conectadas compartiendo datos en tiempo real.
  • Big Data: Uso de grandes volúmenes de datos para optimizar procesos.
  • Inteligencia Artificial (IA): Sistemas que aprenden y mejoran automáticamente.
  • Robótica Avanzada: Robots autónomos y colaborativos (cobots).
  • Impresión 3D: Fabricación aditiva para prototipos rápidos.
  • Ciberseguridad: Protección de sistemas interconectados.
  • Computación en la Nube: Acceso remoto a datos y software.
// CLASE 2: COMPONENTES DEL SISTEMA
Elementos de un sistema de automatización industrial
Elementos que Conforman un Lazo de Control
Los sistemas de control de procesos productivos son "Sistemas de lazo cerrado", ya que buscan automatizar el proceso, mantenerlo dentro de los parámetros exactos y que el sistema se autorregule.
  • Controlador: Generalmente un PLC (equipo electrónico programable).
  • Conversores: Conversores de corriente a presión para válvulas.
  • Actuadores: Ejecutan las acciones en el proceso productivo.
  • Transmisor: Acondicionan las señales eléctricas a un rango estándar.
  • Sensor: Mide la variable del proceso y genera señal eléctrica.
Sensores de Medición Electrónicos
Están en contacto con la variable física del proceso y utilizan o absorben energía del medio controlado para dar una indicación en respuesta a la variación de la variable controlada. El efecto producido puede ser un cambio de presión, fuerza, posición o medida eléctrica.
Ejemplo: En los sensores de temperatura tipo termopar se presenta una variación de voltaje de salida proporcional a la temperatura medida.
Actuadores
El elemento final de control va montado dentro del proceso y recibe la señal del controlador. El actuador EJECUTA la acción de control, logrando regular y acercarse al valor deseado (set point).
Tipos de actuadores: Motores eléctricos, bombas, calefactores, válvulas reguladoras de flujo, válvulas solenoides, variadores de frecuencia, pistones neumáticos o hidráulicos.
Controladores Industriales
Son equipos electrónicos avanzados con capacidad de ser "Programables". Esto permite crear o modificar una rutina según las necesidades del proceso. Posee una entrada donde ingresa la medición de la variable controlada (enviada por transmisores), la compara con el set point y ejerce una acción correctiva según la desviación.
Estructura de Funcionamiento del PLC
El PLC necesita elementos externos para interactuar con el proceso. Solo recibe y envía señales para mantener el proceso regulado al SET POINT (SP).
INPUT: Los sensores que aportan información del sistema se conectan a las ENTRADAS (I en Siemens, X en Delta).
OUTPUT: Los elementos que ejecutan acciones se conectan a las SALIDAS (O en Siemens, Y en Delta).
Elementos de Entrada Digital
Envían señales ON-OFF (TODO-NADA). Solo dos señales: ON=24Vdc (1 lógico) y OFF=0Vdc (0 lógico). Ejemplo: Contacto NA cerrado = 1 lógico.
Elementos de Entrada Analógica
Envían señales CONTINUAS en el tiempo dentro de un rango estandarizado (0-10V o 4-20mA). La señal varía según la variable censada.
Elementos de Salida Digital
Reciben señales ON-OFF del PLC. Cuando activan, el relé conmuta cerrando contactos enviando 24Vdc o 220Vac al actuador.
Elementos de Salida Analógica
Reciben señales CONTINUAS del PLC (4-20mA, 0-10V) variando proporcionalmente la acción de control. El PLC solo envía señales de baja potencia.
Tratamiento de Señales Analógicas
Las señales analógicas (continuas) son estandarizadas por el transmisor a 4-20mA típicamente y enviadas al PLC, donde se convierten internamente a valores de CUENTAS. Por ejemplo, un rango de 4-20mA se convierte a 0-26748 cuentas (según resolución). Esto dependerá del fabricante del PLC.
// CLASE 3: TABLERO ELECTROMECÁNICO
Componentes de control de motores trifásicos
Alimentación: Red Trifásica
Los motores eléctricos industriales son generalmente motores trifásicos. Un sistema trifásico está formado por tres corrientes alternas monofásicas de igual frecuencia y amplitud, con una diferencia de fase de 120° entre ellas. Las fases se designan como R, S y T.
Voltaje Fase-Neutro: 220V AC
Voltaje Fase-Fase: 380V AC
Motor de Inducción Trifásico
Los motores asíncronos o de inducción son motores de corriente alterna en los que la corriente eléctrica del rotor necesaria para producir torsión es inducida por inducción electromagnética del campo magnético del estator (no usa carbones o escobillas).
Historia: El primer prototipo fue desarrollado por Nikola Tesla y presentado en 1888.
Conexiones Estrella y Triángulo
Un motor eléctrico tiene tres bobinados en el estator. Cada bobinado tiene dos extremos:
  • Primer bobinado: U-X (U1-U2)
  • Segundo bobinado: V-Y (V1-V2)
  • Tercer bobinado: W-Z (W1-W2)
Las bobinas pueden configurarse en estrella (arranque suave, menor corriente) o triángulo (mayor potencia, mayor corriente).
Contactor Trifásico
Mecanismo cuya misión es cerrar contactos para permitir el paso de corriente cuando la bobina (A1-A2) recibe corriente eléctrica, comportándose como electroimán.
  • Contactos principales (L-T): Abren o cierran circuito de fuerza que alimenta al motor.
  • Contactos auxiliares (13-14, 53-54): Para circuito de mando, luces y enclavamientos. Soportan menos intensidad.
  • Bobina (A1-A2): Puede ser alimentada en continua o alterna (12V, 24V, 220V).
Módulo de Contactos Auxiliares
Se acopla al contactor para proporcionar más contactos NO y NC. Se monta sobre el contactor con anclaje mecánico (no tiene bobina propia, el enclavamiento es mecánico).
Contactos NO: Segundo número 3 y 4 (Ejemplo: 13-14)
Contactos NC: Segundo número 1 y 2 (Ejemplo: 11-12)
Relé de Sobrecarga (Relé Térmico)
Elemento de protección del motor que desconecta el circuito cuando la intensidad de corriente supera la nominal del motor (debe calibrarse previamente). Evita daños por sobrecorriente, fallo de fase y diferencias de carga excesivas.
Contactos auxiliares: 95-96 y 97-98 cambian de estado al detectar sobrecorriente, apagando el motor y encendiendo luz de advertencia.
Disyuntor Magnetotérmico (Automático)
Dispositivo que interrumpe la corriente eléctrica cuando la intensidad sobrepasa un valor determinado o se produce un cortocircuito. Protege a todo el circuito y equipos conectados.
Cortocircuito: Camino sin resistencia que genera arco eléctrico.
Sobrecarga: Intensidad consumida supera la calibrada (demasiadas cargas).
Protección Diferencial
Dispositivo electromagnético que interrumpe inmediatamente la corriente cuando detecta una fuga de corriente, protegiendo principalmente a las personas de descargas eléctricas. Mide corriente de entrada y salida; si hay diferencia, corta el suministro.
Puesta a Tierra
Sistema de protección que conecta partes metálicas de la instalación eléctrica a tierra mediante conductor y barra/malla de cobre. Crea un camino de baja resistencia para corrientes de fuga, protegiendo personas y equipos de daños, choques eléctricos e incendios.
Pulsadores y Selectores
Pulsador rojo: Detener máquina (no queda enclavado).
Pulsador verde: Activar máquina (no queda enclavado).
Pulsador amarillo: Cambios de giro.
Parada de emergencia: Detiene todo (con enclavamiento).
Luces Piloto
Rojo: Máquina detenida o falla.
Verde: Funcionamiento normal.
Azul: Tensión de línea presente.
Amarillo: Funcionamiento anormal o precaución.
Sensores Industriales
Límites de carrera: Interruptor de posición por contacto físico. Tiene contactos NC (1-2) y NA (3-4). Desventajas: baja velocidad, desgaste mecánico, rebotes.
Sensores inductivos: Detectan metales hasta 60mm sin contacto ni desgaste. Alta frecuencia de conmutación, resistentes a vibraciones, polvo y humedad.
Sensores capacitivos: Detectan materiales conductores y no conductores (vidrio, cerámica, plástico, madera, líquidos). Se usan para identificación, conteo y control de nivel.
Presostatos: Abren o cierran contactos según presión de aire, aceite o agua. Se usan en grupos de presión de agua.
Sensores fotoeléctricos: Responden a cambios en intensidad de luz. Emisor y receptor; cuando el haz se interrumpe, el sensor conmuta. Para detección, clasificación y posicionado de objetos.
Grados de Protección IP
Clasificación que indica la resistencia de un equipo al polvo y agua según norma IEC.
Primer dígito: Protección contra sólidos (1-6).
Segundo dígito: Protección contra líquidos (1-8).
Mayor número = mayor protección.
// CLASE 4: SIMBOLOGÍA Y PLANOS
Interpretación de planos de control con relés
Simbología Normalizada
Para representar sistemas de control de motores eléctricos trifásicos existen 3 normas principales que entregan información clara y sencilla para diseño e implementación:
  • Norma IEC (Europea): Usada internacionalmente.
  • Norma NEMA (Americana): Común en América.
  • Norma JIS (Japonesa): Usada en Asia.
Esta normalización crea un sistema de comunicación gráfico a nivel nacional e internacional.
Elementos de un Sistema de Control de Motores
Los sistemas de control se dividen en dos partes principales:
Circuito de Fuerza: Línea que alimenta al motor trifásico y controla la potencia. Se compone de alimentación trifásica 380V (R, S, T), disyuntor, contactor, relé térmico y motor.
Circuito de Control: Trabaja con elementos de baja potencia (220V o 24V), seguros para manipulación. Compuesto por contactos de mando: pulsadores, paradas de emergencia, sensores y contactos auxiliares.
Consideraciones de los Equipos
Contactor: Posee 3 contactos de fuerza + contactos auxiliares + bobina. Estos actúan en el circuito de control identificados con el número del contactor (KM1, KM2, KM3).
Disyuntor: Opera cuando hay exceso de corriente o cortocircuito, cortando energía en las líneas del circuito de fuerza. No participa en control con contactos auxiliares.
Relé térmico: No opera directamente en fuerza, sino en control mediante contactos 95-96 (corta energía a bobina del contactor) y 97-98 (enciende luz amarilla de falla).
Circuito de Fuerza - Partida Directa
  • Líneas trifásicas (R, S, T): Alimentación 380V entre fases, 220V fase-neutro.
  • Disyuntor magnetotérmico: Protege contra sobrecorrientes y cortocircuito (obligatorio).
  • Contactor: Abre o cierra contactos de fuerza permitiendo corriente al motor. Comandado por bobina A1-A2 en circuito de control.
  • Relé térmico: Protege al motor de excesos de corriente mediante contactos 95-96 y 97-98 en control.
  • Motor trifásico: Carga del circuito. Motor de inducción con 3 bobinas configuradas en estrella o triángulo.
Circuito de Control - Partida Directa
  • Línea de alimentación: L1 y neutro, entrega 220V a bobina del contactor y luces piloto (a veces 24V).
  • Contacto 95-96 (relé térmico): Se abre en exceso de corriente cortando energía al contactor.
  • Contacto 97-98 (relé térmico): Se cierra encendiendo luz amarilla de falla.
  • Pulsador rojo (21-22): Contacto cerrado; al pulsarlo corta energía a bobina deteniendo motor.
  • Pulsador verde (13-14): Contacto abierto; al pulsarlo cierra permitiendo paso de corriente a bobina KM1.
  • Bobina del contactor (A1-A2): Al energizarse, cierra contactos de fuerza arrancando motor.
  • Contacto auxiliar KM1: Mantiene energizada la bobina (enclavamiento) cuando motor arranca.
  • Luz piloto verde: Indica motor funcionando.
  • Luz piloto roja: Indica motor detenido.
  • Luz piloto amarilla: Indica falla térmica.
Aplicaciones Básicas de Control
1. Partida directa mediante botonera START/STOP: Control básico con pulsador de partida (S2) y detención (S1). Protección con fusibles (Q1) y relé térmico (F1). Luces pilotos verde (arranque) y amarilla (falla).
2. Partida con detención temporizada: Similar al anterior pero con temporizador KT1. Al energizarse cuenta hasta valor calibrado; al alcanzarlo abre contacto 55-56 desconectando motor automáticamente.
3. Inversión de giro mediante pulsadores: Con un pulsador gira horario, con otro gira antihorario. IMPORTANTE: motor debe estar completamente detenido antes de cambiar giro para evitar daños.
4. Arranque estrella-triángulo: Sistema complejo con 3 contactores. KM1 da partida, KM3 configura en estrella (arranque suave), luego de 10 segundos KM3 se desactiva y KM2 activa pasando a triángulo (plena potencia).
Nota importante: Se solicita a los estudiantes dedicar tiempo necesario a comprender el funcionamiento de cada circuito, ya que cada uno tiene una lógica de trabajo distinta y son la base para análisis y diseños futuros.
